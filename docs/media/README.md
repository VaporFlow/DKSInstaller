# User guide media assets

Place user-guide screenshots and GIFs in this folder.

## Required filenames

- `01-main-window.png`
- `02-source-and-paths.gif`
- `03-install-action.gif`
- `04-completion-summary.png`

These names are referenced directly from `USER_GUIDE.md`.

## Capture recommendations

- Use PNG for static screens.
- Use short GIFs (8-20 seconds) for flows.
- Capture at 100% display scale when possible.
- Redact personal usernames/paths before publishing.
- Keep image width around 1400-1900px for readability on GitHub.
